# Novel Downloader
一个基于python后端的小说下载器网站项

目前仍在开发，还请多提建议
## 项目功能
用于下载特定网站上的小说为epub格式文件。
## 运行
使用python3运行main.py文件
### 运行库
 - flask
 - urllib
 - random
 - re
 - threading
 - requests
 - lxml
 - ebooklib

有些是标准库，无需安装。
## 注意事项
项目基于python后端

项目仍处于开发阶段，对于安全问题概不负责

项目为网站，暂无线上demo，如需使用，请自行搭建

项目的爬虫部分可单独使用
